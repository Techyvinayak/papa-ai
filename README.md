PapaAI — What Will Papa Say?

An AI-powered web app that predicts exactly how your Indian Papa would react to your life decisions. PapaAI creates a personalized AI version of your Papa.

Users answer a few fun MCQ questions about their father. Based on these answers, the AI generates realistic Papa responses in Hinglish for any situation.

Tech Stack:
- Next.js
- Tailwind CSS
- Framer Motion
- Groq / Gemini API
- React
